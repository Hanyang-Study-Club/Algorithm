- 가보자고~!

## 한양 여왕 코딩 클럽 규칙

- 각자 Github 아이디 명으로 브랜치를 생성한다.
- 각 주차 폴더에 자신의 Github 아이디로 폴더를 생성한다.
- 폴더 안에 코드를 올린다. 파일명은 `연습문제번호_백준문제번호_사용언어` 순으로 작성한다. 
  - ex : `A_1421_python.py`
- 커밋후 PR을 올린다. PR 은 아래와 같은 형식으로 작성한다.
  - 커밋 규칙 : `N주차 아이디`
